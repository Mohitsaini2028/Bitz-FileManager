Bitz File-Manager

File Manager -> that actually manages your files!
Bitz File-Manager is Free to use utility application that can manage all your files and group them together on the basis of file type! It store all files of same type in a seperate directory so that it would be easy for you to find a file if you know what type of file you are looking for like it's a image file or its a pdf, etc. It save your Time and make helps to increase productivity.
